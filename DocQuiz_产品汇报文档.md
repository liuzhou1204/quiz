### 1.1 共情与洞察

**我们是如何发现这个机会的？**

创始团队本身在备考项目管理大赛时，手头有多份 Word 格式的题库资料，想把它们变成可以在线刷题的页面，却遇到了三个障碍：

1. 市面上的题库平台（猿题库、粉笔等）是封闭系统，**不接受用户上传私有题库**
2. 用腾讯问卷等通用工具凑合做，**没有随机出题、错题本、知识点归类等题库逻辑**
3. 找人开发一个系统，**沟通成本高、周期长、费用贵**

这不仅是"我们"的问题。我们通过小范围访谈（15位用户）和问卷（87份）验证了这个痛点的普遍性。

---

### 1.2 三类核心目标用户画像

| 用户类型 | 典型画像 | 核心场景 | 深层需求 | 当前痛点 |
|----------|----------|----------|----------|----------|
| **企业培训负责人** | 中型公司 HR / 培训经理，30-40岁，非技术背景 | 新员工入职考核、岗位技能认证、安全培训 | "我手里有题库，但要找 IT 排期 3 周才能上线" | 依赖内部技术资源，完全无法自主操作 |
| **高校教师 / 辅导员** | 大学讲师、辅导员，28-50岁，基本不写代码 | 期末复习题库、竞赛训练、随堂测验 | "我想让学生在线练习，但我只会 Word" | 只能用问卷工具凑合，体验差，无法统计 |
| **备考社群运营者** | PMP/软考/职业资格备考群主，25-35岁，技术感知尚可 | 考证社群刷题、学习打卡、模拟考试 | "群里有题库 PDF，但我没法变成一个可刷题的工具" | 大平台封闭，不接受私有题库上传 |

### 2.1 方案描述：DocQuiz 是什么？

**DocQuiz** 是一款 **"文档 → 在线题库"** 的自动化工具平台。

核心价值主张只有一句话：

> **上传 Word / PDF，10 分钟拥有你自己的在线题库。**

---

### 2.2 产品三步核心流程

```
┌──────────────┐       ┌──────────────────┐       ┌──────────────────┐
│  ① 上传文档   │  ──→  │  ② AI 自动解析    │  ──→  │  ③ 一键发布分享   │
│              │       │                  │       │                  │
│  Word / PDF  │       │  识别题目类型     │       │  生成在线练习页面  │
│  拖拽上传     │       │  提取选项与答案   │       │  支持随机出题      │
│  支持批量     │       │  兼容多种排版格式  │       │  答题统计         │
│              │       │  自动归类标签     │       │  生成分享链接     │
└──────────────┘       └──────────────────┘       └──────────────────┘
```

**现有 MVP 已实现**：
- `docx_to_json.py` — 自动解析 Word/PDF，提取结构化题目（支持单选、多选、判断）
- Vue 3 在线练习页面 — 多题库切换、答题统计、随机出题
- GitHub Pages 免费托管 — 零服务器成本，即时访问
- 已部署上线：https://liuzhou1204.github.io/quiz/

---

### 2.3 创新类型定位：颠覆性创新矩阵

依据克里斯坦森（Clayton Christensen）的颠覆性创新理论：

| 创新维度 | 大题库平台<br>（猿题库 / 粉笔） | 通用问卷工具<br>（腾讯问卷 / 金数据） | **DocQuiz**<br>（我们的方案） |
|----------|:---:|:---:|:---:|
| **目标客户** | 个人备考者 | 通用场景 | **培训负责人、教师、社群运营者** |
| **性能指标** | 题量大、功能全 | 灵活但不够专业 | **10分钟上手，零技术门槛** |
| **成本结构** | 会员订阅制 | 基础免费 | **开源免费 / 低价订阅** |
| **内容开放度** | ❌ 封闭平台 | —— | **✅ 100% 私有题库** |
| **分发方式** | 应用商店 / 广告投放 | 企业推荐 | **工具口碑传播 · 社群裂变** |
| **用户控制权** | 平台控制内容 | 仅限表单功能 | **用户完全自主** |

**创新类型判断：低端颠覆式创新（Low-end Disruption）**

- 大平台"过度服务"——功能堆叠导致使用门槛高、价格贵
- 我们从一个被忽视的市场底部切入：**那些"够用就行、自己控制内容、不想花钱"的用户**
- 以极简体验 + 极低成本打入市场，随着能力提升逐步向上蚕食

---

### 2.4 为什么具有颠覆性？

核心竞争力集中在一个点上：

> **用 AI 解决了"内容格式转换"这个最大阻力。**

这件事在**技术上并不难**，但在**产品层面此前从未有人专门做好过**。大题库平台不想做（与封闭模式冲突），问卷工具没动力做（不在核心场景）。这就是我们的机会窗口。

### 3.3 产品核心优势（差异化定位）

#### 优势 1：唯一的"文档→题库"自动化方案

在整个市场扫描中，**没有发现任何一个竞品提供"上传 Word/PDF → 自动生成在线题库"的功能**。这是我们的第一壁垒。

#### 优势 2：用户完全拥有内容控制权

与猿题库、粉笔等封闭平台不同，DocQuiz 的内容 100% 属于用户。用户上传的题库、生成的数据都在自己的可控范围内——对企业和学校来说，这是决策的关键因素。

#### 优势 3：极致的零门槛体验

从"我有一个 Word 题库文档"到"学生开始在线刷题"，只需要三步操作。不需要写代码、不需要搭建服务器、不需要申请预算。

#### 优势 4：面向"被忽视的用户群"

我们服务的不是个人刷题者（那是红海），而是**想给别人出题的人**——培训经理、教师、社群运营者。这是一个被大平台系统性忽视的蓝海市场。
